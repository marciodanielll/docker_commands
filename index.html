Comandos Úteis do Docker e Docker Compose

Copiar Comando Descrição
docker container rm $(docker container ls -a -q) -f deleta todos os containers de uma única vez
docker image rm $(docker image ls -q) deleta todas imagens de uma única vez
docker network rm $(docker network ls -q) deleta todas as redes de uma única vez
docker volume rm $(docker volume ls -q) deleta todos os volumes de uma única vez
docker container ls -a lista todos os containers.
docker container ls lista todos os containers em execução
docker container ls --format '{{.Names}}' lista apenas os nomes dos containers que estão em execução
docker info útil para verificar se o serviço daemon está em execução
docker-compose up -d cria e inicia os containers definidos no docker-compose.yml liberando o terminal
docker-compose start inicia os containers listados no docker-compose.yml
docker-compose stop para todos os containers listados no docker-compose.yml
docker-compose down para e remove todos recursos criados dos container listados no docker-compose.yml
